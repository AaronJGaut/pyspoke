# Installation

## Using pip

Install from pip:

```
pip install pyspoke
```

## From latest source

Building the distribution:

```
git clone https://gitlab.com/samflam/pyspoke.git
cd pyspoke
make
```

Installing:

```
make install
```

or

```
pip install dist/*.whl
```

# Testing

From the top level, do:

```
make test
```
